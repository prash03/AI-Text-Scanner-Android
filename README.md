<h1 align="center">🤖 AI Text Scanner (Android)</h1>

<p align="center">
🚀 <b>AI-powered Android app</b> to extract text from images using <b>ML Kit</b> & <b>CameraX</b>
</p>

<hr/>

<h2>🔥 Problem Statement</h2>

<p>In real-world scenarios, users often need to:</p>

<ul>
<li>📄 Copy text from physical documents</li>
<li>📷 Extract information from images</li>
<li>⌨️ Avoid manual typing</li>
</ul>

<p><b>Challenges with traditional methods:</b></p>
<p>
• Time-consuming<br/>
• Error-prone<br/>
• Not scalable
</p>

<hr/>

<h2>💡 Solution</h2>

<p>This app uses <b>on-device AI (ML Kit)</b> to solve the problem:</p>

<ul>
<li>✅ Scan text directly using camera</li>
<li>✅ Instant text extraction</li>
<li>✅ Works offline (no backend required)</li>
<li>✅ Fast, secure & privacy-friendly</li>
</ul>

<hr/>

<h2>🚀 Features</h2>

<ul>
<li>✨ AI-based Text Recognition</li>
<li>📷 CameraX Live Preview</li>
<li>⚡ Real-time Text Extraction</li>
<li>📋 Copy & Share Text</li>
<li>📱 Clean UI (Jetpack Compose)</li>
<li>🧠 On-device AI Processing</li>
</ul>

<hr/>

<h2>🛠 Tech Stack</h2>

<ul>
<li><b>Kotlin</b> – Core language</li>
<li><b>Jetpack Compose</b> – UI</li>
<li><b>CameraX</b> – Camera handling</li>
<li><b>ML Kit</b> – AI Text Recognition</li>
<li><b>MVVM</b> – Architecture pattern</li>
</ul>

<hr/>

<h2>🏗 Architecture</h2>

<p>
UI (Compose)<br/>
⬇️<br/>
ViewModel<br/>
⬇️<br/>
Repository / ML Helper<br/>
⬇️<br/>
ML Kit Processing
</p>

<hr/>

<h2>⚙️ How It Works</h2>

<ol>
<li>📷 Open Camera</li>
<li>📸 Capture Image</li>
<li>🤖 Process using ML Kit</li>
<li>📄 Extract Text</li>
<li>📤 Copy / Share</li>
</ol>
